— LinkedIn canonicalizes URN → vanity
Public URL resolution — done. The URN embedded in the Sales Nav URL (segment between /lead/ and the first comma) is a valid LinkedIn member URN. Pasting it under /in/ yields a URL LinkedIn 301-redirects to the member's vanity profile. LinkedIn Matched Audiences canonicalizes to the member URN server-side, so URN-form URLs match against the same record. Backfilled via scripts/rebuild-public-urls.ts — zero API cost.
